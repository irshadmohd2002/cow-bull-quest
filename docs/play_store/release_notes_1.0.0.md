# Release notes — version 1.0.0

Status: **completed draft**.

## Concise version (Play Console "release notes" field)

Use this in the Play Console release-notes box for the first closed
testing release. Kept short since Play Console release notes are commonly
truncated on some surfaces.

```
First test release of Cow Bull Quest, a Bulls and Cows word-guessing
puzzle. Choose 4-, 5-, or 6-letter words, pick Easy/Common/Hard difficulty,
and crack the secret word using Bull and Cow clues within a limited number
of attempts. Includes light, dark, and system themes, local statistics,
and fully offline play.
```

## Longer, tester-facing version

Use this in the tester-facing announcement (opt-in page, email, or Google
Group post), alongside `docs/play_store/tester_instructions.md`.

```
Welcome to the first public test release of Cow Bull Quest!

Cow Bull Quest is a word-deduction puzzle: guess a secret word, and each
guess earns a Bull for every letter that's correct and in the right
position, and a Cow for every letter that's correct but in the wrong
position. Use those clues to narrow down the answer before you run out of
attempts.

What's in this release:
- Secret words of 4, 5, or 6 letters, each with its own attempt limit
- Three difficulty levels — Easy, Common, and Hard — affecting how
  familiar the secret word is
- Bulls-and-Cows feedback after every guess
- Light, dark, and follow-system theme options
- Local, on-device statistics: wins, losses, streaks, and a breakdown by
  word length and difficulty, with the option to clear them at any time
- Fully offline play — no account, no internet connection required

This is an early test build, so please try a range of word lengths and
difficulties, and let us know what you find using the feedback template
in the tester instructions. Thanks for helping us test Cow Bull Quest!
```
